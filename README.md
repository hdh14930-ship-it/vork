# GitHub User Finder

**Автор:** Иван Иванов

## Описание

Приложение с графическим интерфейсом для поиска пользователей GitHub и сохранения их в избранное.

## Как использовать

1. Введите имя пользователя или часть имени в строке поиска.
2. Нажмите кнопку "Найти".
3. Выберите нужного пользователя из выпадающего списка.
4. Нажмите "Добавить в избранное" — выбранный пользователь сохранится в файл `favorites.json`.
5. Ваши избранные всегда отображаются в нижнем списке.
